# 电影推荐系统简介 python494

## 项目概述

本项目是一个基于Python的协同过滤算法的电影推荐系统，包含了电影播放、广告、会员管理等功能，旨在为用户提供个性化电影推荐服务。

## 技术栈

- 后端技术：Python、Django框架
- 前端技术：Vue.js
- 开发工具：PyCharm
- 数据库：SQLite

## 功能模块

### 前台管理

- 电影信息展示
- 电影资讯浏览
- 个人中心（含会员功能）
- 电影搜索
- 电影收藏
- 电影评分
- 电影播放
- 广告播放

### 后台管理

- 个人中心管理
- 电影分类管理
- 用户管理
- 电影评分管理
- 系统管理
- 电影上传
- 广告上传

## 系统角色

- 普通用户：可以浏览电影、评分、收藏电影等
- 会员用户：拥有普通用户的所有权限，并可享受会员特有服务
- 管理员：负责系统的后台管理
